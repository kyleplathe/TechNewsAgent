## Notes

- **Learning Services LLC** in `local_businesses.ts` matches **Learning Designs Learning Services LLC** (`designedtolearn.com`); MapQuest lists **2822 W 43rd St** (same strip as other services — confirm suite if you cite the address on air).
- **Breadsmith:** Franchise sites list Twin Cities retail as Edina, Minnetonka, and St. Paul (`breadsmithmn.com`). If your on-air list means the **Upton bakery**, many directories show **Great Harvest** at **4314 Upton Ave S** — reconcile naming with a quick sidewalk check before plugs.
- **Pinwheels and Play Toys** uses its **Facebook page** as the main public web presence.
- **Sebesta Apothecary** is strong online; physical studios/partners vary — the map query uses brand + Minneapolis for a sensible card.
